# Autonomous-Cars-Communication

Description:

I implemented an autonomous car communication system using Python, so different cars can communicate with each other and share this type of information:
- Car information (manufacturer, model, unique ID)
- Current Speed
- Current Location (in coordinates X and Y)
- Encountered events in the last 100km

Features:
- Send and Receive information between different cars
- Set current speed
- Calculate distance between cars
- Pick the closest car

How to run: 
- install Python 3.x (I used built-in modules (import math))
- run main.py
